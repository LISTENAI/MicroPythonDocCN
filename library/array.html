    array – 数值数据数组¶

    如下所述，模块实现了相应 CPython 模块的子集。 有关更多信息，请参阅原始 CPython 文档： array.

    支持的格式码： b, B, h, H, i, I, l, L, q, Q, f, d （最后两种取决于浮点型适配）

    类¶

    class array.array(typecode[, iterable])¶

    使用给定类型的元素创建数组。数组的初始内容由 iterable 提供。否则创建一个空数组。

    append(val)¶

    将新元素 val 附加到数组的末尾以增长数组。

    extend(iterable)¶

    将 iterable 中包含的新元素附加到数组的末尾以增长数组。
