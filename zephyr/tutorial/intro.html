    1. MicroPython 的 Zephyr 适配端起手式¶

    让我们开始吧！

    1.1. 依赖¶

    要使用 MicroPython 的 Zephyr 适配端，你将会需要一个 Zephyr 支持的开发板（查看 Zephyr 适配信息概要 了解可用的板子列表）。

    1.2. 上电¶

    如果您的开发板上有一个 USB 接口，那么它很可能在连接到您的 PC 时通过它供电。否则，你需要直接为其供电。请参阅您的开发板的文档了解供电的详细信息。

    1.3. 获取并烧录固件¶

    第一步你需要从 MicroPython 代码仓 克隆项目或者从 MicroPython 下载页面 中下载代码。如果你是一个 MicroPython 的最终使用者，那么从稳定版的固件构建版本开始是一个更推荐的选择。如果你想要在开发版本上工作，那么你可以跟着 git 仓库的日常构建版本来工作。

    接下来，照着 Zephyr 适配端的 readme 文档 (ports/zephyr/README.md) 来编译并在开发板上运行该程序。
